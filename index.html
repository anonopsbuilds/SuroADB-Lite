SuroADB Lite

A Simple ADB (Android Debug Bridge) GUI for Windows.

DOWNLOAD (SourceForge) View on GitHub

Features

Some notable things SuroADB Lite has to offer (there are more under the hood!).

Interactive Buttons

SuroADB Lite uses a plugin that adds and enables button interaction.

Syntax Ready

With the combination of variables and buttons, you can easily customize your preferences for commands such as screenrecord.

Automated Commands

SuroADB Lite does the commands for you- only requiring minimal inputs for everything else.

Simple Menus

Fear not, the sacred texts will lead you to safety (just be extra sure to follow them).

SuroADB Lite 2.4

Be noted that the uninstaller will not close automatically. You may close it manually.

- Fixed the uninstall script (Now exits automatically)

- Updated ADB to platform-tools_r30.0.3-windows
